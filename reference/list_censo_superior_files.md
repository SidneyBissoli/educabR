# List available Higher Education Census files

Lists the data files available in a downloaded Higher Education Census.
Useful for exploring the contents of the ZIP file.

## Usage

``` r
list_censo_superior_files(year)
```

## Arguments

- year:

  The year of the census.

## Value

A character vector of file names found.

## See also

Other Higher Education Census functions:
[`get_censo_superior()`](https://sidneybissoli.github.io/educabR/reference/get_censo_superior.md)
